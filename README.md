# 生生 (SenSen)

> **让每个人拥有无数个自己**
> 
> *我生我，生生不息。*

## 项目简介

生生（SenSen）是一个致力于让每个人都能创造、训练、使用无数个数字分身的平台。这些分身不仅仅是工具，更是基于“硅基灵魂”概念打造的、拥有用户思维方式和价值观的**另一个你**。

我们相信，在信息过剩的时代，人类需要的不是更多的信息，而是能够代替自己去处理信息、去感受世界、去协作共创的数字分身。

## 目录结构

本项目采用 Monorepo 风格管理（虽然目前是多文件夹结构），包含以下主要模块：

- **`documents/`**: 项目核心文档、理念阐述及设计文档。
  - [硅基灵魂概念](documents/生生 · 硅基灵魂.md)
  - [功能介绍](documents/生生 · 功能介绍.md)
- **`sensen-www/`**: 生生官网与产品介绍页。基于 Vue 3 + Vite + TypeScript 开发。
- *(未来将添加更多开发项目，如后端服务、AI 模型训练服务等)*

## 快速开始

如果你想运行官网项目（sensen-www）：

```bash
cd sensen-www
npm install
npm run dev
```
